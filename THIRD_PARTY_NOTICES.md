## 🇨🇳 简体中文

为了让我的世界 Minecraft 玩家只安装一个 Mod JAR 就能使用订阅，本项目内置 Mihomo Windows x64 兼容版。📦🌐

- 🧩 组件：`mihomo-windows-amd64-compatible.exe`
- 🏷️ 版本：`v1.19.28`
- 👥 上游：`MetaCubeX/mihomo`
- 🔗 源代码：https://github.com/MetaCubeX/mihomo/tree/v1.19.28
- 🚀 发行页：https://github.com/MetaCubeX/mihomo/releases/tag/v1.19.28
- 📜 许可证：GNU General Public License v3.0
- 🔐 SHA-256：`a3799f2d75c623a7c6d307e1faf88269e24dd746c59df3e9f1c84d5cfbff6c92`

完整许可证位于 `LICENSES/mihomo-GPL-3.0.txt`。内置文件未经修改，由 Minecraft Mod 自动释放并作为独立子进程运行；游戏退出后自动停止。🧹🛡️

## 🇬🇧 English

The Minecraft mod bundles the compatible Windows x64 Mihomo `v1.19.28` executable so users only install one JAR. It is unmodified, runs as an owned child process, and stops when Minecraft exits. The upstream source and release are linked above. License: GNU GPL v3.0. Full text: `LICENSES/mihomo-GPL-3.0.txt`. 📦⚖️

## 🇷🇺 Русский

Мод Minecraft включает неизменённый Mihomo `v1.19.28` для Windows x64, чтобы пользователю был нужен только один JAR. Mihomo запускается как дочерний процесс мода и останавливается при выходе из Minecraft. Исходный код и выпуск указаны выше. Лицензия: GNU GPL v3.0. Полный текст: `LICENSES/mihomo-GPL-3.0.txt`. 📦⚖️
